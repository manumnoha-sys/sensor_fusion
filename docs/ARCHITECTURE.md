## Component Details

| Component | Details |
|-----------|---------|
| Camera sensor | ON Semi AR1335 вҖ” 13MP, 1/3.2" CMOS, Auto Focus |
| ISP companion | ON Semi AP1302 вҖ” full ISP, I2C addr `0x3C` on mux ch.0 |
| Camera interface | IAS connector (J2/J3), MIPI CSI-2 2-lane |
| FPGA bitstream | `kv260-smartcam` (2022.1) вҖ” loaded via `xmutil` |
| MIPI RX | `xlnx,mipi-csi2-rx-subsystem-5.0` @ `0x80000000` |
| AXI I2C | `xiic-i2c` @ `0x80030000` (i2c-3 on PS) |
| I2C mux | PCA954x @ `0x74`, 4 channels (i2c-4 вҖҰ i2c-7) |
| Video capture | `isp_vcap_csi` вҶ’ `/dev/video0` |
| CPU | ARM Cortex-A53 quad-core, 1333 MHz, aarch64 |
| RAM | 3.8 GiB |
| OS | Ubuntu 22.04.5 LTS (Jammy Jellyfish) |
